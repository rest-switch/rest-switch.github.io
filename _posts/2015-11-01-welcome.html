A security-first open source project utilizing off-the-shelf commodity hardware.

This project aims to create an Internet connected ecosystem of reliable commercial-grade hardware. Security is at the forefront of this project as a key goal.

Project A140808 creates an Internet connected relay array and state sensor array suitable for a wide array of automation tasks. The project includes both web and Android command and control clients.

Near-term project goals include:

  • Internet Thermostat
  • Alarm System Integration
  • Internet Lawn Sprinkler System
  • iOS Client
  • Integration of WeMo hardware
